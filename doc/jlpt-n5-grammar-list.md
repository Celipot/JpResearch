# JLPT N5 — Liste des règles grammaticales

## Particules

- は (topic)
- が (sujet grammatical)
- を (objet direct)
- に (direction, heure, lieu d'existence)
- で (lieu d'action, moyen)
- と (et exhaustif, avec)
- や (et non exhaustif)
- へ (direction)
- から (depuis)
- まで (jusqu'à)
- も (aussi)
- の (possession, nominalisation)
- より (comparaison)
- だけ (seulement)
- ね / よ (fin de phrase)

## Verbes — formes

- Présent/futur poli affirmatif : ます
- Présent/futur poli négatif : ません
- Passé poli affirmatif : ました
- Passé négatif : ませんでした
- Forme て
- Forme た (passé informel)
- Forme nai-kei (ない)
- Forme volitive : ましょう

## Structures de phrases

- A は B です (identification)
- ましょうか (offre d'aide)
- ませんか (invitation polie)
- たいです (vouloir)
- てください (demande polie)
- ないでください (interdiction polie)
- ている (action en cours / état résultant)
- ことができます (capacité)
- に いきます/きます (aller pour faire)
- が すきです / きらいです (goûts)
- が じょうずです / へたです (compétence)
- から (cause/raison)
- が、でも (opposition)
- A より B のほうが (comparaison)
- いちばん (superlatif)
- はどうですか (opinion / proposition)
- のです / んです (explication)

## Adjectifs

- Conjugaison des adjectifs en い (présent, négatif, passé, passé négatif)
- Exception いい → よい
- Conjugaison des adjectifs en な

## Existence

- あります vs います

## Questions

- Forme interrogative en か
- Mots interrogatifs : なに、だれ、どこ、いつ、なぜ、どう、どれ、どちら、どんな、いくら、いくつ

## Démonstratifs こそあど

- これ / それ / あれ / どれ
- この / その / あの / どの
- ここ / そこ / あそこ / どこ
- こちら / そちら / あちら / どちら

## Connecteurs

- そして (et puis)
- それから (et ensuite)
- でも (mais, informel)
- しかし (mais, formel)
- だから (donc, informel)
- それで (donc, neutre)
- または (ou bien)
