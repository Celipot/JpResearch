# Verb Conjugation — Acceptable Answers

All conjugated forms accept both kanji and hiragana input (e.g. 食べます · たべます).

## Forms

| Kind | Dimensions |
|------|-----------|
| Indicative | tense × polarity × register |
| Potentielle | tense × polarity × register |
| Passive | tense × polarity × register |
| Causative | tense × polarity × register |
| Impérative | polarity × register |
| Forme en て | polarity |
| Forme volitionnelle | register |
| Conditionnelle en たら | polarity |
| Conditionnelle en ば | polarity |

## Forme potentielle

- Ichidan : stem + られる → 食べられる
- Godan : e-row + る → 飲める, 書ける
- する → できる
- くる → こられる

Conjugue ensuite comme ichidan. Kanji + hiragana acceptés.

## Forme passive

- Ichidan : stem + られる → 食べられる
- Godan : negative stem + れる → 飲まれる, 書かれる
- する → される
- くる → こられる

Conjugue ensuite comme ichidan. Kanji + hiragana acceptés.

## Forme causative

- Ichidan : stem + させる → 食べさせる
- Godan : negative stem + せる → 飲ませる, 書かせる
- する → させる
- くる → こさせる

Conjugue ensuite comme ichidan. Kanji + hiragana acceptés.
